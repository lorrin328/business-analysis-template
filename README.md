# 经营分析看板 - 太平人寿网电多元条线

**太平人寿保险有限公司网电多元条线** 保险经营分析单页看板。

> 📚 **完整文档体系**：本 README 描述当前实现现状。架构演进、模块化方案、新模块需求、数据规范请见 [docs/](./docs/README.md)。

## 项目概览

本项目定位为太平人寿网电多元条线经营分析看板/经营驾驶舱，当前版本为 **v1.0.97**，采用 **FastAPI + SQLite + 原生 HTML/JS + ECharts** 的轻量架构。核心目标是保证数据处理准确、指标口径统一、页面展示清晰、多人访问稳定，并为后续扩展预留空间。

本项目覆盖以下业务线：

| 业务线 | 说明 |
|--------|------|
| **OTO** | 线上到线下融合业务 |
| **证保** | 证券渠道保险业务 |
| **蚁桥** | 蚂蚁渠道桥梁业务（网服） |
| **经代** | 线上保险经纪/代理业务 |

**当前重点**：正在推动 **职域营销** 新模式，主要与 **团险渠道** 合作开展，目前处于 **试点阶段**。

## 核心模块

### 1. KPI 概览
- 期交保费达成率（整体 / 经代 / 转型 / OTO / 证保 / 蚁桥）
- 价值达成率
- 活动率（同比）
- 人均保费 / 人均产能

### 2. 业务分析
- **业务平台趋势**：支持年度/季度/月度维度，多业务系列复选加总，三种保费口径切换；季度、月度统一按日累计趋势展示，并加入上一年度同期同口径趋势线

### 3. 产品与交期结构
- **产品结构**：饼图展示产品分类占比，并以表格列示 OTO、证保、蚁桥、经代四类业务模式中期交保费占比最高的前三名产品及其模式内占比
- **交期结构**：按交期分类展示保费、保费占比、件数、件数占比，保留图表并增加表格化明细，便于经营复核

### 4. 机构维度
- 支持机构复选、年度/季度/月度切换
- 按机构 × 业务模式展示期交保费、价值保费、长险期交（年度）、10年期产品、商保年金、保障类产品
- 实际、目标、达成率、同比均按所选期间同口径计算

### 5. 队伍分析
- **队伍趋势**：人力规模、活动率、人均保费、人均产能
- 支持年度/季度维度，OTO/证保/蚁桥复选
- **队伍结构与产能分析**：在队伍趋势下方展示司龄段结构、产能段结构、P25/P50/P75 分位数、分位阈值达标人数、分机构分位数和近 6 个月分位趋势；支持年度、季度、月度和业务模式（全部、OTO、证保、蚁桥）切换；口径以 `hr_data` 人员月度原始表为样本，按人员和同一年月左关联 `performance` 保单明细，保留零/负产能人员，避免产能分布被高估

1. 目标设置功能已合并入 `经营分析模板.html`，不再维护独立 `目标设置.html`。

## 开发说明

### 业务口径映射

Excel 原始口径 → 看板口径：

| Excel 口径 | 看板口径 |
|-----------|---------|
| 证券 | 证保 |
| 网服 | 蚁桥 |
| OTO | OTO |

### 保费类型定义

- **期交保费**：非趸交的长期险规模保费 + 短期险规模保费 + 长险趸交规模保费 × 0.1
- **规模保费（年化规保）**：各类产品的规模保费
- **折算保费**：长险趸交 × 0.1 + [3年,5年) × 0.3 + [5年,10年) × 0.5 + 10年及以上 × 1 + 短险 × 0.1

### 指标计算

- **活动率** = 活动人力 / 在职人力 × 100%
- **人均保费** = 月均新单保费 / 月均在职人力
- **人均产能** = 新单保费 / 月均活动人力

```bash
ghcr.io/lorrin328/business-analysis-template:latest
```

运行示例：

```bash
docker run -d \
  --name business-analysis \
  --restart unless-stopped \
  -p 45679:45679 \
  -e BUSINESS_ANALYSIS_DB=/data/business_data.db \
  -v business-analysis-data:/data \
  -v business-analysis-logs:/app/backend/logs \
  ghcr.io/lorrin328/business-analysis-template:latest
```

详细部署、环境变量、数据持久化和回滚说明见 [docs/DOCKER.md](./docs/DOCKER.md)。
