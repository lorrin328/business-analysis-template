# 经营分析看板 - 太平人寿网电多元条线

**太平人寿保险有限公司网电多元条线** 保险经营分析单页看板。

> 📚 **完整文档体系**：本 README 描述当前实现现状。架构演进、模块化方案、新模块需求、数据规范请见 [docs/](./docs/README.md)。

## 项目概览

本项目为太平人寿网电多元条线提供经营分析可视化看板，覆盖以下业务线：

| 业务线 | 说明 |
|--------|------|
| **OTO** | 线上到线下融合业务 |
| **证保** | 证券渠道保险业务 |
| **蚁桥** | 蚂蚁渠道桥梁业务（网服） |
| **经代** | 线上保险经纪/代理业务 |

**当前重点**：正在推动 **职域营销** 新模式，主要与 **团险渠道** 合作开展，目前处于 **试点阶段**。

## 核心模块

### 1. KPI 概览
- 期交保费达成率（整体 / 经代 / 转型 / OTO / 证保 / 蚁桥）
- 价值达成率
- 活动率（同比）
- 人均保费 / 人均产能

### 2. 业务分析
- **业务平台趋势**：支持年度/季度/月度维度，多业务系列复选加总，三种保费口径切换
- **产品结构**：饼图展示各产品分类占比

### 3. 队伍分析
- **队伍趋势**：人力规模、活动率、人均保费、人均产能
- 支持年度/季度维度，OTO/证保/蚁桥复选

## 数据流程

```
Excel 基表（4个文件）
    ↓  Python 解析 + 聚合
SQLite 数据库（business_data.db）
    ↓  JSON 导出
生产 HTML 嵌入（platformMock + teamMock）
    ↓  浏览器渲染
ECharts 图表
```

### 数据源

| 文件名 | 内容 | 行数 |
|--------|------|------|
| `AI-经营分析业绩基表_*.xlsx` | 转型业务保单明细（OTO/证保/蚁桥） | ~69,000 |
| `AI-经营分析价值基表_*.xlsx` | 价值保费数据 | ~140 |
| `N1AI-人力基表_*.xlsx` | 人力数据（在职/活动人力） | ~2,500 |
| `经代业绩分析.xlsx` | 经代渠道业绩数据 | ~125,000 |

### 数据更新步骤

1. 将新的 Excel 文件放到项目根目录
2. 运行 Python 脚本重新聚合：
   ```bash
   python aggregate_data.py
   ```
3. 脚本会自动更新 `data.json`，然后手动将其嵌入 `经营分析模板.html`
4. （可选）更新 `business_data.db` 供 SQLite 查询使用

## 目标设置

打开 `目标设置.html`（浏览器中直接双击打开）：

1. 选择目标年份
2. 在表单中填写各指标的目标值（单位：万）
3. 点击「保存到本地」存储到浏览器 localStorage
4. 点击「导出 JSON」生成备份文件

支持指标：
- 期交保费
- 价值保费
- 商保年金
- 保障类产品
- 10 年期产品

每个指标分年度/季度/月度三个维度，支持整体/经代/转型/OTO/证保/蚁桥六个口径。

## 开发说明

### 业务口径映射

Excel 原始口径 → 看板口径：

| Excel 口径 | 看板口径 |
|-----------|---------|
| 证券 | 证保 |
| 网服 | 蚁桥 |
| OTO | OTO |

### 保费类型定义

- **期交保费**：非趸交的长期险规模保费 + 短期险规模保费 + 长险趸交规模保费 × 0.1
- **规模保费（年化规保）**：各类产品的规模保费
- **折算保费**：长险趸交 × 0.1 + [3年,5年) × 0.3 + [5年,10年) × 0.5 + 10年及以上 × 1 + 短险 × 0.1

### 指标计算

- **活动率** = 活动人力 / 在职人力 × 100%
- **人均保费** = 新单保费 / 月均在职人力
- **人均产能** = 新单保费 / 月均活动人力
