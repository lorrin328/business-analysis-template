# v1.0.81 发布说明

## 发布类型

功能增强版本：队伍结构与产能分析新增标准人力贡献分析。

## 核心变更

1. 在“队伍结构与产能分析”模块新增“标准人力贡献分析”子模块。
2. 支持整体、业务模式、机构、机构 + 业务模式四类维度查看标准人力。
3. 支持分月趋势查看标准人力数量、标准人力占比、期交保费贡献和贡献占比。
4. 后端 `/api/team-enhanced-analysis` 新增 `standardManpower` 数据结构，前端只负责展示。

## 计算口径

- OTO 标准人力：某外勤当月月末在职，且当月折算保费/标准保费大于等于 2 万元。
- 证保标准人力：某外勤当月月末在职，且当月折算保费/标准保费大于等于 3 万元。
- 标准人力保费贡献：标准人力对应人员当月期交保费之和。
- 标准人力占比：标准人力人数 / OTO、证保月末在职人数。
- 保费贡献占比：标准人力期交保费贡献 / OTO、证保全部月末在职人员期交保费。
- 季度/年度口径：按所选月份人月口径汇总；月度口径为当月人数。
