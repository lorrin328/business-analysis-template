# v1.0.61 发布说明

## 一、版本信息

- 版本号：v1.0.61
- 发布日期：2026-05-26
- 类型：队伍分析 / 指标口径 / 数据准确性

## 二、核心变化

本版本将“队伍结构与产能分析（试运行）”从占位展示升级为真实数据计算。新增 `/api/team-enhanced-analysis` 接口，以 `hr_data` 人员月度原始表作为人员样本，按人员代码和同一年月左关联 `performance` 保单明细，输出司龄段结构、产能段结构、P25/P50/P75 分位数和近 6 个月趋势。

## 三、口径说明

1. 默认样本：月初或月末在职人员，保留零/负产能人员。
2. 人员匹配：`hr_data.人员代码 = performance.人员工号`，并统一处理 `1001` 与 `1001.0` 这类数字型工号格式。
3. 时间匹配：同一年、同一月内匹配，不跨月归集。
4. 产能口径：人员月产能 = 当月个人期交保费 / 10000，单位为万元。
5. 分位数口径：P25/P50/P75 按人员月产能升序后线性插值计算，零/负产能人员纳入默认样本。
6. 业务模式映射：证券=证保，网服=蚁桥，OTO 保持不变。
7. 经代口径：经代当前无队伍人力基表，不进入本模块。

## 四、验证结果

- 本地真实库抽查：2026 年 5 月默认样本 468 人，正产能 190 人，零/负产能 278 人，P50 为 0.00 万，P75 为 3.00 万。
- 自动化测试：补充队伍增强分析口径测试、API meta 测试和前端静态测试。

## 五、影响范围

- 影响模块：队伍结构与产能分析（试运行）。
- 不影响模块：KPI 概览、机构维度、业务平台趋势、产品结构、交期结构、目标设置、参数设置、Excel 导出、原队伍趋势图。
