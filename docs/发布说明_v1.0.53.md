# v1.0.53 发布说明

发布日期：2026-05-25

## 一、发布结论

本版本修正机构维度期交保费同比合计口径。修复前，页面只保留本年有实绩的机构/业务模式行，导致去年同期有业绩但本年为 0 的行未进入同比分母，机构维度合计同比被显示为 `+2.0%`。修复后，机构维度合计同比与后端同口径抽数一致，约为 `-1.5%`。

## 二、口径确认

1. 2025 年蚁桥（网服）未归属机构的业绩不纳入机构维度统计范围。
2. 已归属机构的去年同期业绩，即使本年对应机构/业务模式为 0，也必须进入同比分母。
3. 机构维度期交同比口径为：当前机构范围内本年期交保费 / 当前机构范围内去年同期同截止日期交保费 - 1。

## 三、主要变更

1. 前端机构维度行生成规则调整为“本年或去年同期任一方有数据均纳入”。
2. 修复广东蚁桥、浙江蚁桥等去年有业绩但本年为 0 的行被漏出同比分母的问题。
3. 机构维度脚本缓存标识更新到 `v1.0.53`。
4. README、CHANGELOG、发布说明同步更新。

## 四、验证结果

- 当前库复核：2026 年机构维度转型期交 `15,809.73` 万。
- 当前库复核：2025 年同口径机构维度转型期交 `16,050.62` 万。
- 当前库复核：同比约 `-1.5%`。
- 全量测试结果：`171 passed`。

## 五、影响范围

- 影响模块：机构维度。
- 不影响模块：KPI 概览、业务分析、交期结构、队伍分析、目标设置、参数设置、Excel 导出。
- 数据库结构：无新增表、无迁移。
