# v1.0.62 发布说明

## 一、版本信息

- 版本号：v1.0.62
- 发布日期：2026-05-26
- 类型：队伍分析 / 分位数人数 / 期间筛选

## 二、核心变化

本版本继续完善“队伍结构与产能分析（试运行）”，在模块内部新增年度、季度、月度筛选，并补充 P25、P50、P75 对应达标人数和分机构分位数表。

## 三、口径说明

1. 月度：按当月个人期交保费计算人员月产能。
2. 季度/年度：按期间内个人累计期交保费计算人员期间产能。
3. 人数去重：季度/年度同一期间内同一人员只计 1 人；机构、司龄、职等取期间最后一个在职月份。
4. 分位人数：`≥P25人数`、`≥P50人数`、`≥P75人数` 表示人员产能达到该分位阈值及以上的人数。
5. 机构表：按机构分别计算样本、零/负产能占比、P25/P50/P75 和对应达标人数。

## 四、影响范围

- 影响模块：队伍结构与产能分析（试运行）。
- 不影响模块：原队伍趋势图、KPI 概览、机构维度、业务平台趋势、产品结构、交期结构、目标设置、参数设置、Excel 导出。
