# v1.0.71 发布说明

发布日期：2026-05-26

## 一、发布结论

本版本按审计风险回复落实生产安全治理：截止日策略与默认年份保持现状；对本地兜底数据、默认管理员密码、测试鉴权绕过、初始截止日展示和原始表读取方式进行收敛，降低生产误读和权限风险。

## 二、主要变更

1. 生产环境禁用本地 seed 数据自动兜底，仅本机开发或显式 `?localFallback=1` 时允许启用，并显示“开发环境：本地兜底数据”状态。
2. 默认管理员初始密码改为读取 `DEFAULT_ADMIN_PASSWORD` 环境变量；首次初始化无管理员且未配置该变量时，后端拒绝创建默认管理员。
3. `AUTH_TEST_BYPASS` 在 `APP_ENV=production` 时强制失效，避免测试绕过开关误开。
4. 顶部数据截止初始值由固定年月改为 `--`，只在服务端数据加载成功后展示真实截止日。
5. 原始表读取改为显式列清单，减少直接 `SELECT *` 对字段变化和后续维护的影响。

## 三、未调整事项

1. 转型与经代截止日策略按现状保留。
2. 默认年份策略按现状保留。

## 四、验证结果

1. 全量后端与前端静态测试通过。
2. 数据质量审计通过。
3. 版本号、README、CHANGELOG 与前端缓存标识已同步至 `v1.0.71`。
