# v1.0.70 发布说明

发布日期：2026-05-26

## 一、发布结论

本版本新增管理员操作日志能力，用于追踪账号注册、登录、重置密码、导入报表、设置目标、导出 Excel、参数设置和权限管理等关键动作，提升系统上线后的可追溯性和管理闭环。

## 二、主要变更

1. 新增 `operation_logs` 数据表，结构化保存操作人、动作类型、对象用户、状态、详情和发生时间。
2. 新增管理员接口 `/api/admin/operation-logs`，仅管理员组可访问。
3. 前端顶部新增“操作日志”按钮，管理员可直接在弹窗中查看最近操作记录。
4. 登录、注册、管理员新增/修改用户、重置密码、导入报表、设置目标、导出 Excel、参数设置均接入日志记录。
5. 明确 OpenClaw 自动导入边界：脚本式导入不受权限影响，接口式导入需携带具备“数据上传”权限账号的登录令牌。

## 三、验证结果

1. 权限与操作日志专项测试通过。
2. 全量测试通过。
3. 版本号、README、CHANGELOG 与前端缓存标识已同步至 `v1.0.70`。
