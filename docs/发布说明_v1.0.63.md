# v1.0.63 发布说明

## 一、版本信息

- 版本号：v1.0.63
- 发布日期：2026-05-26
- 类型：队伍分析 / 界面筛选

## 二、核心变化

本版本在“队伍结构与产能分析（试运行）”模块内部新增业务模式筛选，支持“全部、OTO、证保、蚁桥”四个选项。

## 三、口径说明

1. “全部”：按 OTO、证保、蚁桥合计计算。
2. “OTO / 证保 / 蚁桥”：仅计算对应业务模式人员样本、产能段、P25/P50/P75、分机构数据和趋势数据。
3. 该筛选只作用于试运行模块，不改变上方队伍趋势图的业务系列复选状态。

## 四、影响范围

- 影响模块：队伍结构与产能分析（试运行）。
- 不影响模块：原队伍趋势图、KPI 概览、机构维度、业务平台趋势、产品结构、交期结构、目标设置、参数设置、Excel 导出。
