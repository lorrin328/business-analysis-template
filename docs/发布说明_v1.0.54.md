# v1.0.54 发布说明

发布日期：2026-05-25

## 一、发布结论

本版本在 KPI 概览“期交保费达成率”卡片小字区域增加经代、转型同比，便于同时查看达成率和业务条线同比表现。转型同比按业务条线口径计算，包含 2025 年蚁桥（网服）未归属机构业绩。

## 二、口径说明

1. 经代同比：本年经代期交保费 / 去年同期经代期交保费 - 1。
2. 转型同比：本年 OTO、证保、蚁桥期交保费合计 / 去年同期 OTO、证保、蚁桥期交保费合计 - 1。
3. 转型同比为业务条线维度比较，包含 2025 年蚁桥未归属机构业绩。
4. 机构维度口径不变：未归属机构业绩不纳入机构维度统计。

## 三、主要变更

1. KPI 卡片小字从“经代达成率、转型达成率”扩展为“经代达成率+同比、转型达成率+同比”。
2. 前端使用服务端 `qj_premium_prev` 数据计算同比，避免从机构维度表反推造成口径混淆。
3. `kpi-cards.js` 增加 `v1.0.54` 缓存标识。
4. README、CHANGELOG、发布说明同步更新。

## 四、验证结果

- 前端静态测试覆盖经代/转型同比取数、展示文本、颜色样式和脚本缓存标识。
- 全量测试结果：`172 passed`。

## 五、影响范围

- 影响模块：KPI 概览。
- 不影响模块：机构维度、业务分析、交期结构、队伍分析、目标设置、参数设置、Excel 导出。
- 数据库结构：无新增表、无迁移。
