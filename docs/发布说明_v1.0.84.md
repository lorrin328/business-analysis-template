# v1.0.84 发布说明

## 发布类型

功能增强版本：权限管理支持删除用户与统一保存。

## 核心变更

1. 管理员组新增删除用户功能。
2. 权限管理弹窗不再为每个用户单独配置保存按钮，改为底部统一保存按钮。
3. 管理员可以批量调整多个用户的角色、密码和模块权限后一次性保存。

## 安全约束

- 不能删除当前登录账号。
- 不能删除最后一个有效管理员账号。
- 删除用户时同步清理该用户会话和模块权限。
- 删除、修改、重置密码仍写入操作日志。

## 验证

- 新增后端权限删除测试。
- 更新前端静态测试，确认统一保存和删除入口存在、行级保存入口不再出现。
- 全量测试通过。
