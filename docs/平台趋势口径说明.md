# 平台趋势口径说明

## 核心结论

平台趋势按真实日累计展示。没有真实日数据时，不生成伪曲线，图表显示“暂无该月份日累计数据”或“暂无该季度日累计数据”。

## 经代日累计

1. 优先检查 `daily_performance` 是否已包含 `channel = 经代`。
2. 如果已包含，经代日累计直接使用 `daily_performance`，不得叠加 `jingdai_daily`。
3. 如果未包含，才从 `jingdai_daily` 补充经代日累计。

## 机构筛选

1. OTO、证保、蚁桥按选中机构过滤。
2. 经代当前暂无机构维度，机构筛选时仍按整体口径展示。
3. 页面在平台趋势图下方提示：“经代暂无机构维度，当前经代数据按整体口径展示。”

## 月份类型

月份统一通过 `normalizeMonth` 转为数字后比较，禁止继续使用宽松相等比较。
