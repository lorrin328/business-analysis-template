# v1.0.59 发布说明

## 版本信息

- 版本号：v1.0.59
- 发布日期：2026-05-26
- 类型：界面优化 / 队伍分析试运行

## 核心变化

本版本继续优化产品结构展示，并在队伍趋势下方新增“队伍结构与产能分析（试运行）”模块。新增队伍模块只读取当前已有汇总数据，不改变现有队伍趋势、KPI、机构维度和数据导入逻辑。

## 主要变更

1. 产品结构表由“每业务模式最高占比 1 个产品”调整为“每业务模式前三名产品”。
2. `/api/product-analysis` 的 `topProducts` 字段新增 `rank`，按 OTO、证保、蚁桥、经代分别返回前三名。
3. 产品结构表格新增“排名”列，展示业务模式、排名、产品名称、期交保费、模式内占比。
4. 队伍趋势下方新增“队伍结构与产能分析（试运行）”模块。
5. 试运行模块展示月均在职、长险活动率、人均保费、人均产能，以及分业务模式汇总表。
6. 司龄段结构、产能段结构、P25/P50/P75、诊断矩阵暂作为待接入项展示，避免在缺少人级/月级底座时误导展示。

## 口径说明

- 产品前三名固定使用期交保费计算。
- 模式内占比 = 单产品期交保费 / 对应业务模式期交保费合计。
- 队伍试运行模块沿用现有队伍趋势数据：月均在职、活动人力、期交保费均来自当前汇总数据。
- 产能分位数和产能段结构暂不计算，需后续新增人级/月级产能表后正式启用。

## 验证重点

- 产品结构后端测试覆盖每业务模式前三名产品和排名。
- 前端静态测试覆盖队伍增强模块存在、原队伍趋势不被替换、关键待接入口径提示存在。
- 本版本无需数据库结构迁移。
