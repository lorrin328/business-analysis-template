# v1.0.68 发布说明

发布日期：2026-05-26

## 一、发布结论

本版本优化“队伍结构与产能分析”模块在手机端的展示效果。该模块包含分位数、分机构、趋势等多列表格，不适合在手机端强行压缩列宽，因此本次改为“摘要卡片自适应 + 表格横向滑动”的展示方式，保证数据完整、排版清晰。

## 二、主要变更

1. 手机端摘要卡片从挤压式排列改为单列/双列自适应展示。
2. 统计期间和业务模式筛选支持换行排列，避免按钮重叠。
3. 队伍结构与产能分析内的表格独立横向滑动。
4. 对长表设置稳定最小宽度，避免 P25/P50/P75、分机构等列被压缩到不可读。
5. 前端缓存标识更新至 `v1.0.68`。

## 三、验证口径

- 前端静态测试覆盖移动端布局规则、模块权限、接口鉴权和缓存标识。
- 全量回归测试确认本次样式调整未影响权限、KPI、导入导出、队伍计算等功能。
