# v1.0.42 发布说明

## 核心修复

修正人均保费的期间口径。

## 正确口径

人均保费不是：

```text
期间累计新单保费 / 月均在职人力
```

而是：

```text
期间月均新单保费 / 期间月均在职人力
```

例如 1-3 月累计人均保费：

```text
(1月新单保费 + 2月新单保费 + 3月新单保费) / 3 / 1-3月月均在职人力
```

这样分子、分母都在月均口径上。

## 本次调整

1. KPI 概览人均保费
   - 年度截至月份：累计转型期交保费先除以统计月数，再除以期间月均在职人力。

2. 人均保费弹窗
   - 年度累计、季度累计、去年同期同比均同步使用月均新单保费口径。
   - 当月口径保持当月新单保费除以当月月均在职人力。

3. 文档与测试
   - 指标定义更新为 `月均新单保费 / 月均在职人力`。
   - 单位明确为 `万元/人/月`。
   - 增加前端静态测试锁定公式。
