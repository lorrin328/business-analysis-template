# v1.0.58 发布说明

## 版本信息

- 版本号：v1.0.58
- 发布日期：2026-05-25
- 类型：界面优化 / 产品结构展示

## 核心变化

本版本优化“业务分析、产品结构、交期结构”的页面层级和展示方式：业务平台趋势单独作为业务分析子模块展示；产品结构与交期结构合并为同一层级的“产品与交期结构”，并补充表格化明细，便于业务侧快速复核结构数据。

## 主要修复

1. 业务平台趋势从原“趋势 + 产品结构”混合布局中拆出，改为独立展示。
2. 产品结构与交期结构放入同一层级，形成并列模块。
3. 产品结构新增“各业务模式最高占比产品”表，覆盖 OTO、证保、蚁桥、经代。
4. 最高占比产品表展示产品名称、期交保费和该产品在本业务模式内的期交保费占比。
5. 交期结构新增表格展示，列示交期分类、保费、保费占比、件数和件数占比。
6. 后端 `/api/product-analysis` 新增 `topProducts` 字段，最高占比产品由后端统一计算，避免前端重复推导。
7. `product-analysis.js`、`payperiod-chart.js` 增加版本缓存标识，避免浏览器继续加载旧脚本。

## 口径说明

- 产品结构最高占比产品固定使用期交保费计算，不受产品结构饼图“期交保费/规模保费”切换影响。
- 转型业务按 OTO、证保、蚁桥三类展示，其中原始表中的“证券”归入证保，“网服”归入蚁桥。
- 经代业务按经代原始表产品名称统计。
- 表格占比公式为：单产品期交保费 / 对应业务模式期交保费合计。
- 交期结构表格占比公式为：单交期分类保费或件数 / 当前筛选范围内合计。

## 验证

- 后端测试覆盖 `topProducts` 计算，验证 OTO、证保、蚁桥、经代四类业务模式的最高占比产品和占比。
- 前端静态测试覆盖模块层级、产品最高占比产品表、交期结构明细表。
- 部署后刷新页面即可生效，无需数据库迁移。
