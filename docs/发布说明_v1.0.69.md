# v1.0.69 发布说明

发布日期：2026-05-26

## 一、发布结论

本版本将“队伍结构与产能分析”的业务模式筛选从单选调整为复选，支持同时选择 OTO、证保、蚁桥中的任意组合，便于查看单一业务线或多业务线合并后的队伍结构与产能表现。

## 二、主要变更

1. 业务模式筛选项调整为“全选、OTO、证保、蚁桥”四个复选框。
2. “全选”勾选时同步勾选 OTO、证保、蚁桥；取消时同步取消三项。
3. 单独勾选/取消 OTO、证保、蚁桥后，模块按当前已选业务线重新计算。
4. 全部业务线均勾选时，不向后端额外传业务模式限制；部分勾选时按 `businessLines` 多值参数传递。
5. 前端缓存标识更新至 `v1.0.69`。

## 三、验证口径

- 前端静态测试覆盖复选框渲染、全选项、空选择和多业务模式传参。
- 全量回归测试确认队伍分析、权限、KPI、导入导出等模块未受影响。
