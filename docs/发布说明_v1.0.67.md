# v1.0.67 发布说明

发布日期：2026-05-26

## 一、发布结论

本版本修复权限管理弹窗中“保存”按钮在部分窗口宽度下不可见的问题，确保管理员调整用户组、重置密码或微调模块权限后，可以直接看到并点击保存。

## 二、主要变更

1. 权限管理表格改为 100% 自适应弹窗宽度。
2. 操作列固定在表格右侧，避免被模块权限列挤出可视区域。
3. 模块权限改为两列展示，减少横向占用。
4. 前端缓存标识更新至 `v1.0.67`。

## 三、验证口径

- 前端静态测试锁定保存按钮样式、右侧固定列和表格自适应宽度。
- 全量回归测试确认权限、队伍结构与产能分析、KPI、导入导出等模块未受影响。
