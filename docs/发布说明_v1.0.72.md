# v1.0.72 发布说明

## 更新主题

新增“星钻联盟 / 荣誉体系”独立业务域底座。

## 主要变化

1. 新增 `honor` 后端业务域，完成字段审计、个人星钻 MVP、批次结果表和异常清单。
2. 新增 `/api/honor/*` 接口，接入现有登录态、模块权限和操作日志。
3. 新增 `/honor` 独立页面，不把星钻逻辑塞入主看板。
4. 新增 `docs/honor_data_assessment.md`，明确现有数据可计算范围和字段缺口。
5. 新增星钻联盟多 Sheet Excel 导出。
6. 新增 `honor_*` 数据表，所有结果保留 `batch_id` 和 `rule_version`。

## 当前边界

1. 经代和蚁桥（网服）均不涉及星钻，不做人级星钻，也不纳入团队星钻。
2. 若 OTO/证保团队内存在网服/蚁桥人员，该人员及其业绩均不计入团队星钻。
3. 团队星钻、证保季度通算和历史批次对账进入后续阶段。
4. 本期不新增星钻专用上传。
