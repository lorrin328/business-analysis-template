# v1.0.60 发布说明

## 版本信息

- 版本号：v1.0.60
- 发布日期：2026-05-26
- 类型：界面优化 / 文案优化

## 核心变化

本版本根据业务反馈优化“队伍结构与产能分析（试运行）”模块展示：删除“诊断矩阵”入口，并将偏技术的“人级底座”表述改为业务可理解的提示。

## 主要变更

1. 删除“诊断矩阵”卡片。
2. 将“待接入人级月度底座”改为“需完善人员月度明细统计”。
3. 将“待接入人级产能分布”改为“需完善人员产能分布统计”。
4. 底部说明改为：司龄段、职等、产能段和分位数需要先完成“每个人每个月”的保费、件数、司龄、职等统计后再正式启用。

## 说明

原“人级底座”的含义是：系统需要先生成每个人每个月的明细统计，包括人员归属、司龄、职等、保费、件数等，才能准确计算司龄段结构、产能段结构和 P25/P50/P75。该表述偏技术，本版本已从页面移除。

## 验证

- 前端静态测试锁定：不再出现“诊断矩阵”和“接入人级底座”表述。
- 本版本不改变数据计算逻辑，不涉及数据库迁移。
