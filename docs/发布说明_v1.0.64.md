# v1.0.64 发布说明

## 一、版本信息
- 版本号：v1.0.64
- 发布日期：2026-05-26
- 类型：权限管理 / 登录认证

## 二、核心变化
本版本新增账号登录与模块权限体系，替代原后台 Token 机制。进入看板前必须登录或注册；系统默认内置管理员账号 `admin`，新注册用户默认为普通用户。

## 三、权限规则
1. 管理员组：拥有全部模块权限，包括权限管理。
2. 高级用户组：除权限管理外，拥有全部模块权限。
3. 普通用户组：默认可查看 KPI、机构维度、业务平台趋势、产品结构、交期结构、队伍分析；默认不可见导入、参数设置、目标设置、Excel 导出和权限管理。
4. 重新计算：按业务确认，所有登录用户均可使用。

## 四、实现说明
- 后端新增用户、会话、用户模块权限表。
- 密码采用加盐哈希保存，管理员只能重置密码，不能查看原密码。
- 导入、参数设置、目标设置、Excel 导出等接口统一改为登录态 + 模块权限校验。
- 前端新增登录/注册拦截、当前用户显示、退出入口和管理员专属权限管理模块。
- 部署预检与健康检查不再要求 `ADMIN_TOKEN`。

## 五、验证结果
- 权限专项与前端静态测试通过。
- 全量回归测试通过：`184 passed`。
