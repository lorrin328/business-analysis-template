## 一、整体架构

单页 HTML Demo，采用深色主题卡片式布局，包含五大区域：

1. **KPI 概览**（6 个卡片）
2. **业务分析**（业务平台趋势 + 产品结构）
3. **队伍分析**（队伍趋势 + 同比分析）
4. **数据上传**（5 个上传入口）
5. **弹窗详情**（点击 KPI 卡片触发）

---

## 二、KPI 概览模块（已完成展示形式）

共 6 张卡片，横向排列。每张卡片可点击，点击后弹出模态框展示明细数据。

### 卡片 1：期交保费达成率（整体达成率）

| 元素 | 内容 |
|------|------|
| 大字 | 期交保费达成率百分比 |
| 上小字 | "期交保费达成率" |
| 下小字 | 分业务模式：OTO / 证保 / 蚁桥 |
| 点击弹窗 | ① 三列达成率卡片（OTO、证保、蚁桥）；② 经代/转型规模数据；③ 堆叠柱状图（分模式月趋势） |

**已实现 mock 数据**：整体 85.3%，OTO 88.2%，证保 90.5%，蚁桥 76.1%

### 卡片 2：价值达成率

| 元素 | 内容 |
|------|------|
| 大字 | 价值达成率百分比 |
| 上小字 | "价值达成率" |
| 右上角 | "需上传" 提示角标 |
| 下小字 | 分 OTO / 证保 / 蚁桥 |
| 点击弹窗 | ① 三列卡片（各模式达成率 + 价值规模）；② 价值规模折线图（分模式） |

**已实现 mock 数据**：整体 72.5%，OTO 75.1%（规模 1,820万），证保 78.3%（规模 1,560万），蚁桥 65.8%（规模 980万）

### 卡片 3：活动率

| 元素 | 内容 |
|------|------|
| 大字 | 活动率百分比 |
| 上小字 | "活动率" |
| 下小字 | "点击查看分模式明细" |
| 点击弹窗 | 表格：OTO / 证保 / 蚁桥 的活动率、环比、在职人力、举绩人力 |

**注意**：线上经代、职域营销暂无活动率指标，弹窗中不显示。

### 卡片 4：商保年金达成率

| 元素 | 内容 |
|------|------|
| 大字 | 商保年金达成率百分比 |
| 上小字 | "商保年金达成率" |
| 下小字 | 经代业务 / 转型业务 |
| 点击弹窗 | 两列卡片：经代达成率 + 转型达成率，含同比变化 |

### 卡片 5：保障类产品达成率

| 元素 | 内容 |
|------|------|
| 大字 | 保障类产品达成率百分比 |
| 上小字 | "保障类产品达成率" |
| 下小字 | 经代业务 / 转型业务 |
| 点击弹窗 | 同卡片 4 格式 |

### 卡片 6：10年期产品达成率（新增）

| 元素 | 内容 |
|------|------|
| 大字 | 10年期产品达成率百分比 |
| 上小字 | "10年期产品达成率" |
| 下小字 | 分 OTO / 证保 / 蚁桥 |
| 点击弹窗 | ① 三列卡片（各模式达成率 + 规模）；② 柱状图（分模式月趋势） |

## 四、图表区域（保持原 demo 结构）

- **业务平台趋势**：折线图，5 条线（OTO、证保、蚁桥、线上经代、职域营销），支持期交/年化规保/折算保费切换
- **产品结构**：环形饼图，支持保费/件数切换
- **队伍趋势**：折线图，支持人力规模/活动率切换
- **同比分析**：柱状图，2025 vs 2026 对比

## 五、待确认事项

1. **达成率口径**：达成率 = 实际 / 目标，目标的颗粒度是到月还是累计？分模式的目标是否需要单独上传？
2. **价值达成率**：价值的计算口径（价值保费、标准保费折算系数）需要核对指标口径手册
3. **活动率分母**：活动率 = 举绩人力 / 在职人力，在职人力是月末时点还是月均？
4. **10年期产品定义**：10年期是指缴费年限 = 10 年，还是保障年限？是否包含 10 年以上的产品？
5. **商保年金口径**：商保年金产品是否包含个人养老金产品？
6. **弹窗图表交互**：是否需要支持口径切换（如弹窗内也支持期交/折算切换）？
