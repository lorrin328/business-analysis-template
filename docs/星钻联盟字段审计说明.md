# 星钻联盟字段审计说明

## 结论

现有数据可以支撑个人星钻 MVP，不需要新增星钻专用上传。

可直接使用的数据：

- `performance`：人员工号、投保单号、业务模式、机构、长短险、折算保费、承保时间、入账时间、回销时间、年月、产品信息。
- `hr_data`：统计年/月、机构、业务模式、人员代码、人员姓名、职等、入职年/月、月末在职、营业组 CODE、营业部 CODE。

适用范围说明：

- 经代和蚁桥（网服）不涉及星钻，不属于星钻字段缺口。
- 字段审计仍列示 `jingdai`，仅用于证明本期未默认新增上传，不代表经代需要纳入星钻计算。

暂不计算：

- 经代和蚁桥（网服）星钻。
- 主管/经理团队星钻。
- 证保季度通算。

完整字段审计报告见 `docs/honor_data_assessment.md`。
