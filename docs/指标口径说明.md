# 指标口径说明

## 核心原则

所有核心指标公式统一放在 `backend/metrics/`，前端不得重复编写经营指标公式。所有除法统一使用 `safe_divide`，自动处理 0、`None`、`NaN`、`Infinity`。

## 指标口径

| 指标 | 公式 | 单位 | 不可计算规则 |
|---|---|---|---|
| 达成率 | 实绩 / 目标 | % | 目标为空或为 0 |
| 同比 | 本期 / 去年同期 - 1 | % | 去年同期为 0 或缺失 |
| 环比 | 本期 / 上期 - 1 | % | 上期为 0 或缺失 |
| 序时进度 | 已过时间 / 总周期 | % | 周期参数缺失 |
| 进度偏差 | 达成率 - 序时进度 | 百分点 | 达成率或序时进度不可计算 |
| 活动率 | 活动人力 / 在职人力 | % | 在职人力为 0 或缺失 |
| 人均保费 | 新单保费 / 月均在职人力 | 万元/人 | 月均在职人力为 0 或缺失 |
| 人均产能 | 新单保费 / 月均活动人力 | 万元/人 | 月均活动人力为 0 或缺失 |
| 转化率 | 成交数 / 触达或线索数 | % | 基数为 0 或缺失 |
| 费用率 | 费用 / 保费 | % | 保费为 0 或缺失 |
| 投产比 | 产出 / 投入 | 倍 | 投入为 0 或缺失 |

## API 返回要求

统一 API 返回中需通过 `meta.metric`、`meta.unit`、`meta.dataSource` 标明指标、单位和数据来源；复杂指标在 `meta.definitions` 中返回口径说明。
