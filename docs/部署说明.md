# 部署说明

## 推荐部署

Ubuntu Server + Nginx + systemd + FastAPI + SQLite。

## 一键部署

```bash
sudo bash deploy/deploy.sh
```

脚本会完成：

1. 安装 Python、Nginx、rsync。
2. 同步项目到 `/opt/business-analysis`。
3. 创建 Python 虚拟环境并安装依赖。
4. 初始化 SQLite 表和索引。
5. 安装 systemd 服务。
6. 安装 Nginx 配置并重启服务。

## 常用命令

```bash
sudo systemctl status business-analysis
sudo journalctl -u business-analysis -f
sudo systemctl restart business-analysis
tail -f /opt/business-analysis/backend/logs/app.log
```

## 数据备份

部署脚本会在覆盖前备份 `/opt/business-analysis/business_data.db` 到 `/opt/business-analysis-backups`。

## Docker 可选路径

```bash
cd deploy
docker compose up -d
```
