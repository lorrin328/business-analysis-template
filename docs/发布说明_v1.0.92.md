# v1.0.92 发布说明：荣誉体系离职清零口径修正

## 一、更新结论

本版本修正荣誉体系“追踪人力”口径：统计月人力基表中不存在的人员，按当月已离职处理，不再计入当月追踪人力，并在星钻计算中自动清零。

## 二、口径调整

1. 当月人力基表不存在：视为当月已离职。
2. 当月人力基表存在但月末在职人力为 0：视为当月非月末在职。
3. 当月已离职或非月末在职人员：不计入当月追踪人力、会员人数、资深及以上人数和测算奖励。
4. 离职/非在职清零：历史追踪人员若当月不在人力基表中，或当月月末在职人力为 0，系统生成当月清零记录，钻石余额清零，会员等级为“未入会”。
5. 年度累计追踪池：保留为辅助指标，用于说明年度内曾进入星钻追踪范围的人数。

## 三、展示调整

- “追踪人力”显示当月月末在职口径。
- 新增“累计追踪池”，显示年度内曾进入追踪范围的人员去重数。
- 新增“离职/非在职清零”，显示统计月因不在人力基表或月末非在职而被清零的人数。
