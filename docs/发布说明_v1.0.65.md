# v1.0.65 发布说明

## 一、版本信息
- 版本号：v1.0.65
- 发布日期：2026-05-26
- 类型：登录注册修复 / 前端缓存修复

## 二、修复内容
本版本修复登录或注册后页面提示 `window.setAuthSession is not a function` 的问题。

## 三、原因说明
浏览器可能继续使用旧版 `api-client.js` 缓存，旧脚本不包含新版账号体系所需的 `setAuthSession` 方法，导致登录/注册接口成功后前端会话写入失败。

## 四、处理措施
- `api-client.js` 与 `auth-ui.js` 增加 `v1.0.65` 缓存标识。
- `auth-ui.js` 增加最小会话函数兜底，即使 API 客户端缓存未及时刷新，也可以完成登录注册。
- 补充前端静态测试锁定该行为。

## 五、验证结果
- 全量回归测试通过：`184 passed`。
