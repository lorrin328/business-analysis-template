# 星钻联盟测试用例

## 字段审计

1. raw 表不存在时返回缺表状态。
2. raw 表存在但缺人员代码时，人级星钻规则不可计算。
3. 缺折算保费时提示可按缴费年限降级复算。
4. 缺团队字段时，不阻断个人星钻。
5. 同义字段可匹配。
6. 输出 A/B/C/D 可计算性分级。

## 规则

1. 初级会员门槛：3 颗。
2. 中级会员门槛：6 颗。
3. 资深会员门槛：12 颗。
4. OTO 达标：折算保费 >= 20000 且长险 >= 1。
5. 证保达标：折算保费 >= 30000 且长险 >= 1。
6. 证保保号：有长险未达标不扣减。
7. 未达标扣减 1 颗。
8. 钻石不扣成负数。
9. 月末非在职清零。
10. 新星人力：当年入职且 3 个自然月内累计 >= 3 颗。

## API

1. 未登录访问 `/api/honor/summary` 返回 401。
2. 无权限访问 `/api/honor/recalculate` 返回 403。
3. `honor_view` 可访问 summary。
4. `honor_audit` 可访问 field-audit。
5. `honor_export` 可导出。
