# v1.0.50 发布说明

## 版本信息

- 版本号：v1.0.50
- 发布日期：2026-05-25
- 类型：ui / display-logic

## 核心变更

1. 机构维度默认按机构汇总展示，降低首屏信息密度。
2. 新增“展开/收起”按钮，展开后展示机构下 OTO、证保、蚁桥等业务模式明细，并保留机构小计。
3. 达成率按年度自然日序时进度与 100% 阈值着色：
   - 低于序时进度：绿色
   - 高于序时但低于 100%：淡红色
   - 大于等于 100%：红色
4. 同比按增速水平着色：
   - 同比为负：绿色
   - 同比为正但低于 10%：淡红色
   - 同比大于等于 10%：红色
5. 机构维度达成率、同比统一保留 1 位小数。

## 影响范围

- 仅影响机构维度模块的展示层逻辑。
- 不改变后端接口、数据库结构、目标口径和指标计算口径。
- 汇总行仍由明细行聚合生成，保证与展开明细一致。

## 验证结果

- 前端静态测试：`27 passed`
- 全量回归测试：`169 passed`

## 部署提示

本版本为前端展示优化，部署代码并刷新页面即可生效。
