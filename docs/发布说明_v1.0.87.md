# v1.0.87 发布说明

## 版本信息

- 版本号：v1.0.87
- 发布日期：2026-05-29
- 类型：功能增强 / AI 只读接口

## 核心结论

本版本新增面向 ChatGPT、自定义 GPT 和外部 AI 工具的专用只读接口。AI 可以通过 `https://kpi.bcyt.tech:30443/api/ai/*` 读取经营看板数据，但不能执行任何写操作。

## 主要变更

1. 新增 AI 只读接口：
   - `/api/ai/dashboard-snapshot`
   - `/api/ai/kpi`
   - `/api/ai/org-summary`
   - `/api/ai/team-summary`
   - `/api/ai/metric-definitions`
   - `/api/ai/openapi.json`

2. 新增独立鉴权机制：
   - 使用 `AI_READONLY_TOKEN`。
   - 支持 `Authorization: Bearer <token>` 或 `X-AI-Token`。
   - 未配置令牌时返回 `503`，避免默认暴露数据。

3. 明确安全边界：
   - 不复用管理员账号。
   - 不开放导入、重算、目标设置、参数设置、权限管理。
   - 不开放数据库直连和任意 SQL。
   - AI 读取动作写入操作日志。

4. 部署支持：
   - systemd 服务新增 `/opt/business-analysis/deploy/.ai_env` 环境文件入口。
   - 新增 `docs/AI只读接口说明.md`。

## 测试

- 新增 AI 接口鉴权测试。
- 新增错误令牌、未配置令牌、快照返回、日志记录和 OpenAPI 公开说明测试。
- 全量测试通过后发布。
