# v1.0.47 发布说明

## 版本信息

- 版本号：v1.0.47
- 发布日期：2026-05-25
- 类型：feature / export

## 核心变更

1. 页面右上角新增“导出Excel”按钮，支持按当前年份导出经营分析看板数据。
2. 新增后端接口 `/api/export/excel`，由服务端基于当前 FastAPI + SQLite 查询结果直接生成工作簿。
3. 导出工作簿按模块拆分 sheet，包括导出说明、KPI 概览、目标设置、机构维度、平台趋势、产品结构、产品明细、交期结构、队伍分析、产品参数。
4. Excel 表格增加标题、深色表头、边框、筛选、冻结窗格、自动列宽和百分比格式，便于直接查阅和二次加工。
5. 当目标为空或服务端目标尚未配置时，导出文件展示明确空状态，不影响其他模块导出。

## 口径说明

- 导出数据复用当前服务端 API 和 SQLite 聚合查询结果，不在前端重新计算核心指标。
- 金额单位沿用看板口径，默认单位为万元。
- 目标数据以 `target_config` 服务端目标为主源。
- 产品参数 sheet 直接读取 `product_config`，用于核对商保年金、保障类产品配置状态。

## 验证结果

- 本地回归测试：`164 passed`
- 新增测试覆盖：
  - Excel 导出接口可正常返回 `.xlsx` 文件。
  - 工作簿可被 `openpyxl` 正常打开。
  - 导出 sheet 名称、表头和基础格式符合预期。
  - 前端导出按钮和 `js/export-excel.js` 纳入生产运行边界。

## 部署提示

本版本只涉及代码、文档和前端入口更新，不需要数据库结构迁移。部署后重启 `business-analysis.service` 即可生效。
