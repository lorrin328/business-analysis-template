# 星钻联盟规则口径说明

## 个人星钻 MVP

适用范围：转型业务中的 OTO、证保。

暂不纳入：经代和蚁桥（网服）均不涉及星钻，不做人级星钻，也不进入团队星钻归集。

## 月度达标

| 业务线 | 月度折算保费 | 长险件数 | 结果 |
|---|---:|---:|---|
| OTO | >= 20,000 元 | >= 1 件 | 获 1 颗钻石 |
| 证保 | >= 30,000 元 | >= 1 件 | 获 1 颗钻石 |

证保当月有长险件但未达标，记为保号月份，不扣减。

未达标且非保号月份扣减 1 颗钻石，最低不低于 0。

月末非在职人员当前钻石清零，会员等级显示未入会。

## 会员等级

累计钻石按最高达成门槛取会员等级。低于 3 颗显示未入会。

| 会员级别 | 累计钻石门槛 |
|---|---:|
| 星曜会员 | 100 |
| 星钻会员 | 84 |
| 恒星会员 | 60 |
| 金星会员 | 48 |
| 至尊会员 | 36 |
| 钻石会员 | 30 |
| 白金会员 | 24 |
| 黄金会员 | 18 |
| 资深会员 | 12 |
| 高级会员 | 9 |
| 中级会员 | 6 |
| 初级会员 | 3 |
| 未入会 | 0 |

## 异常处理

字段缺失、业绩归属使用年月字段降级、负数保费、人员工号缺失等情况均写入 `honor_exceptions`，不得静默计算。

## 团队口径

团队星钻后续实施时，只归集 OTO、证保适用人员。若某 OTO 或证保团队内存在网服/蚁桥人员，该人员及其业绩均不计入团队星钻；经代同样不纳入星钻团队口径。
