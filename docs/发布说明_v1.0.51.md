# v1.0.51 发布说明

发布日期：2026-05-25

## 一、发布结论

本版本在机构维度新增“长险期交（年度）”指标，补齐目标、达成、达成率三项展示。计算口径沿用已确认规则：长险期交达成率使用期交保费目标作为分母，实绩取转型业务长险期交年度累计。

## 二、主要变更

1. 后端 `/api/org-analysis` 新增 `longterm` 数据块，按机构和业务模式返回长险期交年度达成。
2. 长险期交实绩来源为 `agg_longterm_qj`，仅纳入转型业务；经代不进入机构维度。
3. 存在日级数据时，长险期交年度累计按渠道截止日截断，避免与机构维度期交保费口径不一致。
4. 前端机构维度表格新增“长险期交（年度）”列组，展示目标、达成、达成率。
5. 前端目标映射将 `longterm` 明确绑定到 `qjPremium`，保证目标口径与业务确认一致。
6. Excel 导出“机构维度”sheet 同步新增长险期交目标、达成、达成率三列。

## 三、验证结果

- 后端新增回归测试：校验长险期交年度值、日级截断、经代排除。
- 前端新增静态测试：校验机构维度列组、目标映射和年度口径。
- Excel 导出测试：校验机构维度 sheet 包含长险期交三项列头。
- 全量测试结果：`171 passed`。

## 四、影响范围

- 影响模块：机构维度、Excel 导出。
- 不影响模块：KPI 概览、业务分析、交期结构、队伍分析、目标设置、参数设置。
- 数据库结构：无新增表、无迁移。
