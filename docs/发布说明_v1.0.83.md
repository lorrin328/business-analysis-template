# v1.0.83 发布说明

## 发布类型

修复版本：队伍结构与产能分析统一月末在职人员底座。

## 核心变更

1. 队伍结构与产能分析主样本由“月初或月末在职过”调整为“月末在职”。
2. 月度口径：仅纳入当月月末在职人员。
3. 季度/年度口径：纳入期间内任一统计月月末在职过的人员，并按人员代码去重。
4. 标准人力贡献分析、顶部样本人数、司龄结构、产能段、分位数和机构分位数统一使用同一月末在职人员底座。

## 口径影响

原 v1.0.82 中，2026年5月选择 OTO+证保时：

- 顶部样本人数为 405 人，包含月初在职但月末不在职的 13 人。
- 标准人力贡献分析为 392 人，仅统计月末在职。

v1.0.83 起，两处统一为月末在职口径，人数应保持一致。

## 验证

- 新增测试覆盖月初在职但月末不在职人员的剔除逻辑。
- 全量测试通过。
- `git diff --check` 通过。
