# v1.0.74 发布说明

## 发布类型

ui / honor / usability

## 核心结论

本版本聚焦“星钻联盟 / 荣誉体系”页面的可读性和经营跟踪效率，不调整星钻计算公式和数据口径。页面从技术明细表展示，调整为更适合经营管理查看的机构对比、人员下钻、预警跟踪和数据适配说明。

## 主要变更

1. 机构对比增加排名、人均钻石和筛选范围经营摘要，便于快速识别会员率领先机构和扣减压力机构。
2. 人员明细增加姓名/代码/机构关键词、机构、业务模式、会员等级、新星人力筛选。
3. 预警清单和异常清单补充人员姓名、保单号、建议动作，并增加机构、类型、等级筛选。
4. 将前台“字段审计”表述调整为“数据适配”，明确其用途是判断现有数据能否支撑星钻计算，而不是经营结果页。
5. 后端异常接口补充人员姓名映射，提升问题定位效率。

## 验证重点

- 主看板“荣誉体系”入口显示正常。
- 荣誉体系页面版本号为 v1.0.74。
- 机构对比、人员明细、预警清单、异常清单均可筛选。
- 预警与异常表包含人员姓名。
- 自动化测试通过后推送 GitHub。
