# v1.0.52 发布说明

发布日期：2026-05-25

## 一、发布结论

本版本修复机构维度“展开/收起”按钮交互不稳定、浏览器可能继续使用旧脚本导致颜色标识不生效的问题。修复后，机构维度默认展示机构汇总，点击“展开”展示 OTO、证保、蚁桥明细，再次点击恢复机构汇总；达成率和同比按既定规则显示颜色。

## 二、主要变更

1. 展开按钮由内联 `onclick` 改为运行时事件绑定，减少脚本加载边界和缓存状态带来的失效风险。
2. 新增按钮状态同步逻辑，维护 `aria-expanded` 与按钮文案，确保默认“展开”、展开后“收起”。
3. 保留机构维度相关全局入口，兼容上传、年份切换等其他模块调用。
4. 机构维度脚本增加版本号缓存标识，避免浏览器继续加载旧版 `org-analysis.js`。
5. 前端静态测试补充按钮绑定、默认状态、颜色类生成和缓存标识断言。

## 三、验证结果

- 前端静态测试：通过。
- 全量测试结果：`171 passed`。
- 浏览器实测：默认 11 行机构汇总；点击展开后 24 行业务模式明细；再次点击恢复 11 行机构汇总。
- 浏览器实测：达成率/同比单元格已生成 `org-ind-green`、`org-ind-light-red`、`org-ind-red` 样式类，并呈现对应颜色。

## 四、影响范围

- 影响模块：机构维度。
- 不影响模块：KPI 概览、业务分析、交期结构、队伍分析、目标设置、参数设置、Excel 导出。
- 数据库结构：无新增表、无迁移。
