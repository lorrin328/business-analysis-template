# v1.0.41 发布说明

## 核心目标

将本次人均保费问题沉淀为自动化数据质量审计，避免后续再依赖人工肉眼发现类似问题。

## 新增能力

1. 数据质量审计服务
   - 检查原始表重复明细。
   - 从 SQLite 原始表重新计算关键聚合，并与当前 `agg_*` 表比对。
   - 检查长险期交是否超过总期交。
   - 检查活动人力、人均类指标分母是否存在明显异常。

2. 命令行工具

```bash
cd backend
./venv/bin/python audit_data_quality.py --year 2026 --json
```

3. 管理诊断接口

```text
GET /api/diagnostics/data-quality?year=2026
```

该接口需要后台管理 Token。
