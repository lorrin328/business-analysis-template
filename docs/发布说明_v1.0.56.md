# v1.0.56 发布说明

## 结论

本版本在 KPI 概览“期交保费达成率”卡片中新增整体同比，便于同时看到整体、经代、转型三层同比表现。

## 主要变化

1. 期交保费卡片小字区域新增“整体同比”。
2. 整体同比使用服务端 `qj_premium_prev.total`，与本年整体实绩 `qj_premium.total` 对应。
3. 展示顺序调整为：整体同比、经代达成率及同比、转型达成率及同比。
4. 经代同比、转型同比、机构维度未归属机构处理规则均保持不变。
5. 同步更新指标计算口径核对报告中的期交保费卡片展示说明。
6. 前端版本号更新为 `v1.0.56`，同步 `kpi-cards.js` 缓存标识。

## 口径说明

```text
整体同比 = 本年（经代期交保费 + 转型期交保费） / 去年同期（经代期交保费 + 转型期交保费） - 1
```

## 验证

- 前端静态测试已覆盖整体同比取数和展示文本。
- 本地全量回归测试：`173 passed`。
