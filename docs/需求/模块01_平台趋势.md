## 4. 计算口径

### 4.1 主图：累计折线

```sql
-- 月度累计示例（折算保费）
SELECT year, month,
       SUM(zhsf_cents) / 100.0 AS amount
FROM fact_premium
WHERE <筛选条件>
GROUP BY year, month
ORDER BY year, month;
```

JS 端按 year 分组后逐月累加得到 cumulative 数组。

### 4.2 同比

```
yoy(year, period) = (current_cum - base_cum) / base_cum × 100%
base_year = new Date().getFullYear()  // 系统当前年份，自动滚动
```

**用户决策（2026-04-26）**：基准年取系统当前年份，**不硬编码**、**不允许用户在 UI 选择**。每年元旦后第一次打开会自动切换基准年。

### 4.3 KPI

每个分组取最后非零期的累计值。

### 4.4 日明细

```sql
SELECT day, SUM(metric_cents) / 100.0 AS amount
FROM fact_premium
WHERE year = ? AND month = ? AND day IS NOT NULL
GROUP BY day
ORDER BY day;
```

JS 端逐日累加为累计折线。缓存键：`${year}|${gran}|${period}|${metric}|${filterHash}`。
