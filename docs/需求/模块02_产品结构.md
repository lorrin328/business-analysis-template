## 4. 计算口径

### 4.1 单年饼图

```sql
SELECT <dim_col> AS k, SUM(<metric_cents>) / 100.0 AS v
FROM fact_premium
WHERE year = ? AND <筛选>
GROUP BY <dim_col>
ORDER BY v DESC;
```

过滤规则：
- `v == 0` 的项过滤（保留负值，即撤单/退保）
- `k = NULL` 或空串 → 显示为「未知」

### 4.2 年度对比（2 饼并排）

并排展示选定的两年（默认基准年 + 当前年）：

```
┌─────────────┬─────────────┐
│  2024年结构 │  2026年结构  │
│  [饼图]    │   [饼图]     │
└─────────────┴─────────────┘
+ 下方表格：每个分类两年金额、占比、变化幅度
```

### 4.3 占比计算

```
占比_i = v_i / SUM(v_j for all j)
变化 = 占比_2026 - 占比_2024  (绝对差)
增速 = (v_2026 - v_2024) / v_2024  (相对)
```
