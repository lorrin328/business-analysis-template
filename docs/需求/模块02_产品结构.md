## 4. 计算口径

### 4.1 单年饼图

```sql
SELECT <dim_col> AS k, SUM(<metric_cents>) / 100.0 AS v
FROM fact_premium
WHERE year = ? AND <筛选>
GROUP BY <dim_col>
ORDER BY v DESC;
```

过滤规则：
- `v == 0` 的项过滤（保留负值，即撤单/退保）
- `k = NULL` 或空串 → 显示为「未知」

### 4.2 年度对比（2 饼并排）

并排展示选定的两年（默认 **基准年 - 1** vs **基准年**，基准年 = 系统当前年份）：

```
┌─────────────┬─────────────┐
│ {当前年-1}年│ {当前年}年   │
│  [饼图]    │   [饼图]     │
└─────────────┴─────────────┘
+ 下方表格：每个分类两年金额、占比、变化幅度
```

> 例：2026 年使用时默认对比 2025 vs 2026；UI 仍允许手动选择不同年份。

### 4.3 占比计算

```
占比_i = v_i / SUM(v_j for all j)
变化 = 占比_当年 - 占比_去年   (绝对差)
增速 = (v_当年 - v_去年) / v_去年   (相对)
```
