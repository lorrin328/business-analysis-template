## 4. 产能定义（用户 2026-04-26 已确认）

### 4.1 三个产能定义

| 编号 | 指标 | 分子 | 分母 | 说明 |
|------|------|------|------|------|
| **P1** | **人均保费** | 期交保费 (qj_cents / 100) | 月均在职人力 | 含全员（含未出单） |
| **P2** | **人均产能** | 期交保费 (qj_cents / 100) | 月均**活动人力** | 仅活动员工人均 |
| **P3** | **人均件数** | 件数（COUNT DISTINCT policy_no，**排除附加险**） | 月均在职人力 | 件均频次指标 |

**关键规则**：
- ✅ 分子统一用「期交保费」（qj_cents），不用折算保费
- ✅ P1 和 P3 分母用「月均在职人力」
- ✅ P2 分母用「月均活动人力」（活动人力 = 当月有保单的员工）
- ✅ P3 件数计算时**必须排除附加险**（识别规则待 Q10 确认）

**注**：「活动人力」定义见模块 04（取代旧称「活跃人力」）。

### 4.3 分母口径

- P1、P3 分母：**月均在职人力** = `(SUM(hc_start) + SUM(hc_end)) / 2.0`
- P2 分母：**月均活动人力** = 当月与前月活动人力数取均值（待精确化，见 Q14）

> ⚠️「月均活动人力」的精确算法需在 P5 实施时确认：
> - 选项 A：`(月初活动人力 + 月末活动人力) / 2`，但活动人力是"月内"概念，月末未必=月初
> - 选项 B：直接取本月活动人力数（不平滑）
> - 默认选项 B（简单），见开放问题 Q14

## 5. 计算口径示例

### 5.1 P1 人均保费

```sql
WITH premium_by_month AS (
  SELECT year, month, org,
         SUM(qj_cents) / 100.0 AS qj_total
  FROM fact_premium
  WHERE <筛选>
  GROUP BY year, month, org
),
hc_avg_by_month AS (
  SELECT year, month, org,
         (SUM(hc_start) + SUM(hc_end)) / 2.0 AS hc_avg
  FROM fact_headcount
  GROUP BY year, month, org
)
SELECT p.year, p.month, p.org,
       p.qj_total,
       h.hc_avg,
       p.qj_total / NULLIF(h.hc_avg, 0) AS per_capita_premium
FROM premium_by_month p
JOIN hc_avg_by_month h USING(year, month, org)
ORDER BY p.year, p.month;
```

### 5.2 P2 人均产能（活动人力分母）

```sql
WITH premium_by_month AS (
  SELECT year, month, org,
         SUM(qj_cents) / 100.0 AS qj_total
  FROM fact_premium
  WHERE staff_id != '' AND zhsf_cents > 0
  GROUP BY year, month, org
),
active_hc AS (
  -- 当月活动人力（有保单的不重复员工数）
  SELECT year, month, org,
         COUNT(DISTINCT staff_id) AS act_n
  FROM fact_premium
  WHERE staff_id != '' AND zhsf_cents > 0
  GROUP BY year, month, org
)
SELECT p.year, p.month, p.org,
       p.qj_total / NULLIF(a.act_n, 0) AS per_active_premium
FROM premium_by_month p
JOIN active_hc a USING(year, month, org)
ORDER BY p.year, p.month;
```

### 5.3 P3 人均件数（不含附加险）

```sql
WITH cnt_by_month AS (
  SELECT year, month, org,
         COUNT(DISTINCT policy_no) AS policy_cnt
  FROM fact_premium
  WHERE policy_no != ''
    AND zhsf_cents > 0
    AND <附加险过滤条件>   -- ⚠️ 待 Q10 确认（见主方案）
  GROUP BY year, month, org
),
hc_avg_by_month AS (
  SELECT year, month, org,
         (SUM(hc_start) + SUM(hc_end)) / 2.0 AS hc_avg
  FROM fact_headcount
  GROUP BY year, month, org
)
SELECT c.year, c.month, c.org,
       c.policy_cnt,
       h.hc_avg,
       1.0 * c.policy_cnt / NULLIF(h.hc_avg, 0) AS per_capita_count
FROM cnt_by_month c
JOIN hc_avg_by_month h USING(year, month, org);
```

> 「附加险过滤条件」临时占位，待 Q10 确认后替换。可能形态：
> - `AND is_pension = 0`（如果附加险都标 is_pension）
> - `AND product_name NOT LIKE '%附加%'`（按产品名）
> - `AND product_code NOT IN (附加险代码字典)`（按字典）
