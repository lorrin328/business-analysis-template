## 一、模块定位

KPI 概览模块是看板的 **首屏核心区域**，承担以下职能：

- **一目了然的经营健康度**：通过 6 张关键指标卡片，让管理层在 3 秒内掌握整体经营态势
- **下钻分析入口**：每张卡片可点击，进入更细粒度的分维度分析
- **数据完整性提示**：通过"需上传"角标，提示缺失数据源

## 七、计算口径说明

### 7.1 达成率通用公式

```
达成率 = 实际累计值 ÷ 目标累计值 × 100%
```

**累计口径**：截至当前月份的本年累计值。例如 4 月查看时，目标 = 1–4 月目标之和，实际 = 1–4 月实际之和。

### 7.2 各指标计算

#### 期交保费达成率

```
期交保费达成率 = SUM(期交保费) ÷ SUM(期交保费目标)
```

维度：可按整体、分业务模式、分渠道（经代/转型）计算。

#### 价值达成率

```
价值达成率 = SUM(价值保费) ÷ SUM(价值保费目标)
```

> **注意**：价值保费口径需与财务部确认，是否为"价值规保"字段，或按产品价值率折算。

#### 活动率

```
活动率 = 本月累计举绩人力 ÷ 月均在职人力 × 100%
```

其中：
- **举绩人力** = 当月有出单且在职的人力（以 `人员工号` 去重）
- **月均在职人力** = (月初在职人力 + 月末在职人力) ÷ 2

> **待确认**：在职人力是取月末时点，还是月均？口径手册定义为月均。

#### 商保年金达成率

```
商保年金达成率 = SUM(商保年金产品期交保费) ÷ SUM(商保年金目标)
```

筛选条件：`is_annuity = 'Y'`（或 `是否商保年金产品 = '是'`）

#### 保障类产品达成率

```
保障类产品达成率 = SUM(保障类产品期交保费) ÷ SUM(保障类产品目标)
```

筛选条件：`design_cat` 属于保障类（长期健康险、长期寿险、短期险），且 `is_annuity = 'N'`

> **待确认**：保障类产品的精确定义范围。

#### 10年期产品达成率

```
10年期产品达成率 = SUM(缴费年限=10年的产品期交保费) ÷ SUM(10年期产品目标)
```

筛选条件：`pay_years = '10年'`（或 `10`）

> **待确认**：是否仅指缴费年限 = 10 年，还是包含 ≥ 10 年？是否仅指长期险？
