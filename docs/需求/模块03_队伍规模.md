## 1. 背景

当前系统仅有保单数据，无人力规模视图。用户已提供 `N1AI-人力基表`，本模块基于该数据源建立队伍规模趋势看板，支持多维度对比。

## 2. 范围

### 2.1 包含

- 月度在职人力趋势（折线 / 柱状）
- 多维度对比：按机构、销售序列、职等、司龄区间
- 月度留存率、增员/减员折线
- 司龄结构占比（堆叠柱 / 饼图）
- 与基准月对比的变化幅度
- 联动全局筛选（机构、业务模式）

### 2.2 不包含

- 个人级别人员档案查询（不展示具体姓名/工号）—— 隐私考虑
- 主管/经理层级聚合（人力表只有营业部/营业组 CODE，没有完整层级树，待开放问题 Q6 决策）
- 招聘漏斗（无数据源）

## 3. 数据需求

来自 `fact_headcount`，详见人力数据规范。

| 字段 | 用途 |
|------|------|
| `period_ym, year, month` | 时间分组 |
| `org, sales_seq, dept_code, team_code` | 组织维度 |
| `rank_grade, rank_level` | 职等/职级维度 |
| `tenure_band` | 司龄分箱 |
| `biz_mode` | 业务模式（联动保单） |
| `hire_year, hire_month` | 入司分析 |
| `hc_start, hc_end` | 月初/月末在职指标 |

## 4. 计算口径

### 4.1 月末在职人力

```sql
SELECT period_ym, org, SUM(hc_end) AS headcount
FROM fact_headcount
WHERE period_ym BETWEEN ? AND ?
  AND <全局筛选>
GROUP BY period_ym, org
ORDER BY period_ym;
```

### 4.2 月初/月末/月均人力（可切换口径）

```sql
SELECT period_ym, org,
       SUM(hc_start) AS hc_start_total,
       SUM(hc_end) AS hc_end_total,
       (SUM(hc_start) + SUM(hc_end)) / 2.0 AS hc_avg
FROM fact_headcount
GROUP BY period_ym, org;
```

### 4.3 留存率

```sql
1.0 * SUM(hc_end) / NULLIF(SUM(hc_start), 0) AS retention_rate
```

> 留存率含义：月末在职 / 月初在职。低于 1 表示净流出，高于 1 表示当月新增 > 流失。
> 严格意义上"留存率"应排除当月新入职：`留存率 = (月末在职 ∩ 月初在职) / 月初在职`。
> v1 先用简化口径，v2 视需求增强。

### 4.4 增员/减员（推算）

由于没有精确入司/离司日期，按月度差额推算：

```sql
-- 月度净变化
SELECT a.period_ym, a.org,
       SUM(a.hc_end) - LAG(SUM(a.hc_end)) OVER (PARTITION BY a.org ORDER BY a.period_ym) AS net_change
FROM fact_headcount a
GROUP BY a.period_ym, a.org;
```

更精细的"新增 vs 流失"：

```sql
-- 当月新增 = 月末在职但上月不存在
WITH curr AS (SELECT period_ym, staff_id, hc_end FROM fact_headcount),
prev AS (SELECT period_ym, staff_id, hc_end FROM fact_headcount)
SELECT c.period_ym,
       SUM(CASE WHEN c.hc_end = 1 AND p.hc_end IS NULL THEN 1 ELSE 0 END) AS hires,
       SUM(CASE WHEN c.hc_end = 0 AND p.hc_end = 1 THEN 1 ELSE 0 END) AS leavers
FROM curr c
LEFT JOIN prev p ON p.staff_id = c.staff_id
                 AND p.period_ym = strftime('%Y-%m', date(c.period_ym || '-01', '-1 month'))
GROUP BY c.period_ym;
```

### 4.5 司龄结构

```sql
SELECT period_ym, tenure_band, SUM(hc_end) AS n
FROM fact_headcount
GROUP BY period_ym, tenure_band
ORDER BY period_ym, tenure_band;
```

司龄区间已经分好（a-f 6 档），直接 GROUP BY。

### 4.6 入司分析

```sql
SELECT hire_year, hire_month, COUNT(DISTINCT staff_id) AS hire_n
FROM fact_headcount
WHERE period_ym = ?  -- 取最新月份
GROUP BY hire_year, hire_month
ORDER BY hire_year, hire_month;
```
